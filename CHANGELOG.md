## [0.1.0] - 2026-04-13

### Added
- CRT barrel distortion with adjustable curvature
- Scanline effect with cosine-based pattern
- Chromatic aberration for RGB channel offset
- Vignette edge darkening
- Random glitch bursts with customizable effects
  - Horizontal shift
  - RGB split
  - Pixel noise
  - Freeze effect
- Support for both Camera2d and Camera3d
- Embedded WGSL shader (no asset files required)
- Interactive 2D example with egui controls
- Interactive 3D example with PBR scene
- Bloom and tonemapping compatibility
- Configurable glitch parameters
- Hot reload support for development
